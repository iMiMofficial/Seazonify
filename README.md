## 🎯 What is Seazonify?

Seazonify is a powerful SaaS platform that enables website owners to add magical visual and audio effects to any website with a single line of code. Create engaging, seasonal experiences that delight visitors and boost engagement—no coding required.

### 🌟 Open Source at Heart

**Our effect library is completely open source!** All effects are available on GitHub for anyone to use, modify, and learn from.

- 📦 **[Browse Effects on GitHub →](https://github.com/iMiMofficial/Seazonify/tree/main/assets/effects)**
- 🔓 Free to use via CDN links
- 📚 Learn from production-ready code
- 🤝 Community-driven development
- ✨ Contribute your own effects

**Use effects directly from GitHub CDN:**
```html
<!-- Use any effect for free via CDN -->
<script 
  src="https://cdn.jsdelivr.net/gh/iMiMofficial/Seazonify@main/assets/controller/v1/controller.js"
  visual="https://cdn.jsdelivr.net/gh/iMiMofficial/Seazonify@main/assets/effects/visual/dist/winter-snowflakes-classic-v1.0.0.js"
  defer crossorigin="anonymous"
></script>
```

**Seazonify Platform** adds powerful features on top of the open-source library:
- 🎛️ Visual workspace for non-developers
- 📊 Analytics and engagement tracking  
- 📅 Smart scheduling and automation
- 🌍 Weather-based triggers
- 🔑 API access for programmatic control
- 🎨 Effect customization UI
- 📈 Performance optimization

### ✨ Key Features

- 🎨 **Audio-Visual Effects** - Professionally crafted visual, audio, and combo effects
- 🔧 **One-Line Integration** - Works on any website (WordPress, Shopify, custom sites)
- 🎮 **Live Preview Workspace** - Test and customize effects before deployment
- 🧪 **Effect Lab** - Create and submit your own custom effects
- 📊 **Advanced Analytics** - Track engagement, views, and performance metrics
- 📅 **Smart Scheduling** - Auto-activate effects based on date, time, season, or weather
- 🌐 **Multi-Site Management** - Control effects across unlimited websites
- 📱 **Fully Responsive** - Optimized for mobile, tablet, and desktop
- ⚡ **CDN Delivery** - Lightning-fast global content delivery
- 💳 **Flexible Billing** - Free tier + scalable Pro plans

## 🚀 Quick Start

### For Website Owners

Add Seazonify to your website in 3 simple steps:

1. **Create Account:** [Sign up for free](https://app.seazonify.com/signup)
2. **Choose Effects:** Browse and preview at [Workspace](https://app.seazonify.com/workspace)
3. **Choose a Plan:** Get a pro plan for Auto-Seazonify. Add your website from Integration page.
4. **Copy & Paste:** If you're a free user, just add one line of code. Copy from under the **Live Preview** window.

```html
<script
  src="https://cdn.jsdelivr.net/gh/iMiMofficial/Seazonify@main/assets/controller/v1/controller.js"
  visual="https://cdn.jsdelivr.net/gh/iMiMofficial/Seazonify@main/assets/effects/visual/dist/autumn-leaves-3d-min-v1.0.0.js"
  defer crossorigin="anonymous"
></script>
```

That's it! Your effects are live. 🎉

**No coding skills required. Works on any website.**

## ❓ FAQ

### General

**Q: Is Seazonify free?**  
A: Yes! We have a free forever plan. Pro plans start at $12/month.

**Q: Does it slow down my website?**  
A: No. Effects are loaded asynchronously and optimized for performance. Average load time: < 100ms.

**Q: Can I use it on multiple websites?**  
A: Yes. But only one effect at a time.

**Q: Do I need coding skills?**  
A: No! Our workspace is completely visual. Coding is only needed if you want to create custom effects.

### Effects

**Q: Can I request custom effects?**  
A: Yes! You can submit request to our community.

**Q: How do I create my own effect?**  
A: Read the [Effect Guidelines](EFFECT_GUIDELINES.md) and use the [Lab](https://app.seazonify.com/lab).

**Q: Will my effect be approved?**  
A: If it follows our guidelines, performs well, and is high quality, yes! Review takes 1-3 business days.

### Technical

**Q: Is there an API?**  
A: Yes! Pro plans include RESTful API access. [API Docs](https://seazonify.com/aura/docs/api)

**Q: What about GDPR/Privacy?**  
A: We're GDPR compliant. We collect anonymized analytics data (views, engagement) to provide insights to website owners. No personal data is collected from visitors. [Privacy Policy](https://seazonify.com/privacy)
