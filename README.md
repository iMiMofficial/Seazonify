### General

**Q: Is Seazonify free?**  
A: Yes! We have a free forever plan. Pro plans start at $12/month.

**Q: Does it slow down my website?**  
A: No. Effects are loaded asynchronously and optimized for performance. Average load time: < 100ms.

**Q: Can I use it on multiple websites?**  
A: Yes. But only one effect at a time.

**Q: Do I need coding skills?**  
A: No! Our workspace is completely visual. Coding is only needed if you want to create custom effects.

### Effects

**Q: Can I request custom effects?**  
A: Yes! You can submit request to our community.

**Q: How do I create my own effect?**  
A: Read the [Effect Guidelines](EFFECT_GUIDELINES.md) and use the [Lab](https://app.seazonify.com/lab).

**Q: Will my effect be approved?**  
A: If it follows our guidelines, performs well, and is high quality, yes! Review takes 1-3 business days.

**Q: Is there an API?**  
A: Yes! Pro plans include RESTful API access. [API Docs](https://seazonify.com/aura/docs/api)

**Q: What about GDPR/Privacy?**
